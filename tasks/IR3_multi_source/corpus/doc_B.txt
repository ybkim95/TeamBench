[10]

[12] - Backend language: TypeScript 5.4
[13] - API framework: Hono 4.2
[14] - Frontend framework: Nuxt 3.11
[15] - Runtime environment: Node 20
[16]
[17] DESIGN DECISIONS
[18] The event-driven architecture was chosen after a formal RFC process.
[19] All inter-service calls use Apache Kafka for type safety and performance.
[20] The CockroachDB datastore supports the required ACID guarantees.
